# 🎨 Bootstrap Color Palette (Responsive)

This is a **beginner-level project** that shows how to create a **responsive color palette** using Bootstrap 5.

---

## 🌐 Live Demo

👉 https://btcolorpallet.ccbp.tech/

---

##  About Project

This project displays different Bootstrap background colors in a **responsive grid layout**.
It adjusts automatically for mobile, tablet, and desktop screens.

---

##  Technologies Used

* HTML
* Bootstrap

---

## Features

*  Responsive design
*  Uses Bootstrap grid system
*  Uses background color utilities (`bg-*`)
*  Clean and simple layout

---
##  Responsive Layout

* Mobile → 1 column
* Small devices → 2 columns
* Medium devices → 3 columns
* Large devices → 4 columns
* Extra large → 6 columns

---

##  Colors Used

* Primary
* Secondary
* Success
* Danger
* Warning
* Info
* Dark
* Light

## 🧠 What I Learned
