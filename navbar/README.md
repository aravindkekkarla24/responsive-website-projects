# 🍔 Food Munch Navbar (Responsive)

This project is a **responsive navigation bar** built using **Bootstrap 5**.
It is part of a food website UI and adapts to different screen sizes (mobile, tablet, desktop).

---

## 🌐 Live Demo

👉 https://navbarfoodmuch.ccbp.tech/

---

## 📌 Features

* ✅ Fully responsive navbar
* ✅ Logo integration
* ✅ Navigation links aligned to right
* ✅ Clean and simple design

---

## 🧱 Technologies Used

* HTML
* Bootstrap
* Custom CSS

---


## 📱 Responsive Behavior

* Large screens → full navbar visible
* Small screens → collapsible menu (hamburger icon)

---

## 🧠 Learning Outcome

* Understand Bootstrap navbar
* Use utility classes like `ms-auto`

---
